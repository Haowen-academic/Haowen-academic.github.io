# Haowen Dai Academic Homepage

Static GitHub Pages site for Haowen Dai.\r\n\r\n## Deploy

1. Create a GitHub repository named `Haowen-academic.github.io`.
2. Copy these files into the repository root.
3. Commit and push to the `main` branch.
4. In GitHub, open `Settings -> Pages` and select `Deploy from a branch`, `main`, `/root`.

The homepage should then be available at:

```text
https://Haowen-academic.github.io
```

## Edit Content

- Main page content: `index.html`
- Visual style: `assets/css/styles.css`
- Animation behavior: `assets/js/main.js`
